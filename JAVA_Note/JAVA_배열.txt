[배열]
 - 선언
	데이터타입[] 변수명 ;
	데이터타입 변수명[];
 - 생성
	변수명 = new 데이터타입[크기];
 - 할당
	변수명[인덱스] = 값;
 - reference type
 - 생성 시 기본값으로 초기화
	정수=0, 문자=\u0000, 논리=false, 참조=null
 - length 속성 : 배열의 크기
	0부터 length-1 번째까지 인덱스 접근 가능
 - 배열의 인덱스를 벗어난 값에 접근하면 error
	IndexOutOfBoundsException


[다차원 배열]
자바의 배열은 모두 1차원.
다차원으로 생성해도 실질적으로는 1차원 여러개를 만들고 각 배열을 참조하는 참조배열을 하나 더 만드는 방식.
